## RareSkills Solidity Beginner Projects

This repository is part of a personal learning journey implementing the **ten beginner Solidity projects** described in the RareSkills article: [Ten beginner project ideas after you learn Solidity](https://rareskills.io/post/beginner-solidity-projects).

The goal is to **consolidate knowledge** from tutorials and exercises, and to **demonstrate practical skills** by building real smart contracts, writing tests, and running basic analysis tools.

---

## Project List

These are the ten projects from the RareSkills article that this repository aims to implement (from easiest to hardest):

1. **Purchase NFT with ERC20 tokens**  
   Build an NFT that can only be minted by paying with a particular ERC20 token.

2. **Time unlocked ERC20 / vesting contract**  
   A payer deposits tokens into a contract, and the receiver can only withdraw `1/n` tokens over `n` days.

3. **NFT Swap Contract**  
   Two users trustlessly swap NFTs by depositing them into a contract that enforces matching addresses and tokenIds.

4. **Crowdfunding ERC20 contract**  
   Create and manage fundraiser campaigns with goals and deadlines; support ETH and ERC20-based crowdfunding.

5. **English auction contract**  
   Auction NFTs with reserve price, deadline, multiple concurrent auctions, and bid withdrawals for non‑winners.

6. **Simple NFT Marketplace**  
   Sellers list NFTs with price and expiration; buyers purchase directly without custodying NFTs in the marketplace.

7. **Stake Together**  
   Users stake a token and receive rewards proportional to their share of the total stake after a fixed period.

8. **Simple lottery**  
   Users buy lottery tickets during a time window; a future blockhash determines the winner, who must claim in time.

9. **ERC1155 Bingo**  
   Use ERC1155 tokens to simulate bingo cards; players mint random numbers and the first to get a line wins.

10. **On-chain blackjack**  
   Fully on‑chain open‑hand blackjack with a dealer, random cards, and enforced turn timing.

Treat all contracts here as training material only. Do not deploy them to handle real value without a professional security review.
